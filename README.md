# rna-classify
Classify RNA structure

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```
cp .env.example .env
```
Fill in the values in the .env file.

```sh
npm install
npm run dev
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Building

```
npm run build
```
